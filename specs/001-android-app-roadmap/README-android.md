# Android App Development Guide

## Overview

This feature transitions the existing `index.html` web app into a Capacitor-wrapped Android application ready for Google Play Store distribution.

## Architecture

- **Wrapper**: Capacitor 5+ for modern Android WebView
- **Web Assets**: Bundled in `www/` directory
- **Package ID**: `com.myapp.id`
- **Permissions**: INTERNET, ACCESS_FINE_LOCATION, ACCESS_COARSE_LOCATION

## Key Files

- `capacitor.config.json`: Capacitor configuration
- `android/`: Native Android project (generated by Capacitor)
- `www/index.html`: Main web app entry point
- `www/css/leaflet.css`, `www/js/leaflet.js`: Bundled Leaflet assets

## Development Workflow

1. Make changes to `www/index.html` and supporting assets
2. Run `npx cap sync android` to update native project
3. Open in Android Studio: `npx cap open android`
4. Build and test on emulator or device

## Mobile Optimizations

- Responsive design with viewport meta tag
- Touch-friendly navigation with 44px+ tap targets
- Bundled critical assets for offline performance
- localStorage persistence for user data

## Deployment

- Use Android Studio to generate signed AAB
- Test with bundletool for local validation
- Submit to Google Play with appropriate metadata

## Testing Checklist

- [ ] App launches without crashes
- [ ] All navigation tabs work
